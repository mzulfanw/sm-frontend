### Getting Started
Make sure you have the following installed: 
- Node 
- ESLint

``` bash
# 1. Clone 
git clone git@github.com:mzulfanw/sm-frontend.git

# 2. Enter to cloned directory
cd sm-frontend

# 3. install all dependencies. Make sure you installed npm 
npm install 

# 4. Run on your local 
npm run dev
